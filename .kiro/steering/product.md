# Product

Algorithms Visualizer is an interactive web app for visualizing classic computer science algorithms. It has two main sections:

- Pathfinding Visualizer: interactive grid where users can place walls and weights, then watch pathfinding algorithms (Dijkstra, A*, BFS, DFS, Greedy BFS, Swarm) find a path from start to finish. Supports maze generation via Recursive Division.
- Sorting Visualizer: animated bar chart showing sorting algorithms (Bubble, Selection, Merge, Quick, Heap Sort) in action with configurable speed and array size.

Deployed as a static site on GitHub Pages at https://minhhung19494.github.io/algo_demo/
